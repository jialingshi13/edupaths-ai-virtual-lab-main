# HKDSE Chemistry Teaching Materials

## Purpose

This file is a teaching-material bank for the Chemistry Virtual Lab. It gives teachers and students the chemical equations, ionic equations, redox equations, electrolysis equations, enthalpy cycles, organic reactions, practical observations and DSE explanation points needed to support the virtual experiments.

## How to Use

For each virtual experiment, show:

- aim
- apparatus
- key equation
- particle explanation
- observation
- graph or data output
- DSE exam point
- common mistake

## Core Equation Writing Rules

### State Symbols

| Symbol | Meaning | Example |
|---|---|---|
| `(s)` | solid | `CaCO3(s)` |
| `(l)` | liquid | `H2O(l)` |
| `(g)` | gas | `CO2(g)` |
| `(aq)` | aqueous solution | `HCl(aq)` |

### Balancing Rules

1. Count atoms of each element on both sides.
2. Balance metals first.
3. Balance non-metals except H and O.
4. Balance H and O near the end.
5. Check charge for ionic and redox equations.
6. Never change formula subscripts to balance equations.

Example:

```text
Mg(s) + HCl(aq) -> MgCl2(aq) + H2(g)

Balanced:
Mg(s) + 2HCl(aq) -> MgCl2(aq) + H2(g)
```

## Chapter 1: Planet Earth and Separation

### Filtration

Teaching equation:

```text
No chemical reaction occurs.
Separation is based on particle size.
```

Key terms:

- residue: solid left on filter paper
- filtrate: liquid passing through filter paper

Virtual lab observation:

- insoluble solid remains on filter paper
- clear filtrate collects in conical flask

DSE focus:

- filtration separates insoluble solid from liquid
- filtration cannot separate dissolved solute from solvent

### Evaporation and Crystallization

Teaching equation:

```text
Salt(aq) -> Salt(s) after solvent evaporation
```

Example:

```text
NaCl(aq) -> NaCl(s)
```

Virtual lab observation:

- solvent level decreases
- crystals appear when solution becomes saturated

DSE focus:

- do not heat to complete dryness when preparing crystals
- crystallization obtains pure soluble solid

### Simple Distillation

Teaching equation:

```text
No chemical reaction occurs.
Separation is based on boiling point difference.
```

Example:

```text
Seawater -> water distillate + salt residue
```

DSE focus:

- thermometer bulb should be near the side arm
- condenser water enters from the lower end and leaves from the upper end

### Chromatography

Rf equation:

```text
Rf = distance moved by spot / distance moved by solvent front
```

Example:

```text
spot distance = 3.2 cm
solvent front distance = 8.0 cm
Rf = 3.2 / 8.0 = 0.40
```

DSE focus:

- baseline must be drawn in pencil
- baseline must be above solvent level
- different substances have different solubility and attraction to paper

## Chapter 2: Atomic Structure and Bonding

### Atomic Structure

Core relationships:

```text
Atomic number = number of protons
Mass number = number of protons + number of neutrons
Charge = number of protons - number of electrons
```

Examples:

```text
Na atom: 11 protons, 11 electrons, charge = 0
Na+ ion: 11 protons, 10 electrons, charge = +1
Cl- ion: 17 protons, 18 electrons, charge = -1
```

Electron arrangement examples:

```text
Na: 2,8,1
Mg: 2,8,2
Cl: 2,8,7
Ar: 2,8,8
Ca: 2,8,8,2
```

DSE focus:

- isotopes have same number of protons but different number of neutrons
- ions form by gain or loss of electrons, not protons

### Ionic Bonding

Electron transfer equations:

```text
Na -> Na+ + e-
Cl + e- -> Cl-
Na+ + Cl- -> NaCl
```

Magnesium oxide:

```text
Mg -> Mg2+ + 2e-
O + 2e- -> O2-
Mg2+ + O2- -> MgO
```

Calcium chloride:

```text
Ca -> Ca2+ + 2e-
2Cl + 2e- -> 2Cl-
Ca2+ + 2Cl- -> CaCl2
```

DSE explanation:

- metal atoms lose electrons to form cations
- non-metal atoms gain electrons to form anions
- oppositely charged ions attract strongly in a giant ionic lattice
- ionic compounds have high melting points
- ionic compounds conduct electricity when molten or aqueous because ions are mobile

### Covalent Bonding

Hydrogen:

```text
H. + .H -> H:H
H2
```

Chlorine:

```text
Cl. + .Cl -> Cl:Cl
Cl2
```

Water:

```text
2H + O -> H2O
O shares one electron pair with each H atom.
```

Carbon dioxide:

```text
C + 2O -> CO2
O=C=O
```

Methane:

```text
C + 4H -> CH4
```

DSE explanation:

- non-metal atoms share electrons
- simple molecular substances usually have low melting and boiling points
- covalent molecules do not conduct electricity because there are no mobile ions or electrons

### Metallic Bonding

Teaching model:

```text
Metal atoms -> positive metal ions + delocalized electrons
```

DSE explanation:

- delocalized electrons conduct electricity
- layers of metal ions can slide, so metals are malleable
- alloys are harder because different-sized atoms distort the regular lattice

```text

```

```text

```

```text
Oxidation Is Loss of electrons

```

```text
Oxidation: oxidation number increases
Reduction: oxidation number decreases
```

Agents:

```text
Oxidizing agent is reduced.
Reducing agent is oxidized.
```

### Common Half-Equations

Zinc oxidation:

```text
Zn(s) -> Zn2+(aq) + 2e-
```

Copper(II) reduction:

```text
Cu2+(aq) + 2e- -> Cu(s)
```

Magnesium oxidation:

```text
Mg(s) -> Mg2+(aq) + 2e-
```

Silver(I) reduction:

```text
Ag+(aq) + e- -> Ag(s)
```

Hydrogen ion reduction:

```text
2H+(aq) + 2e- -> H2(g)
```

Hydroxide oxidation:

```text
4OH-(aq) -> O2(g) + 2H2O(l) + 4e-
```

Chloride oxidation:

```text
2Cl-(aq) -> Cl2(g) + 2e-
```

### Balancing Redox Equations in Acidic Medium

Method:

1. Split into oxidation and reduction half-equations.
2. Balance atoms except O and H.
3. Balance O using H2O.
4. Balance H using H+.
5. Balance charge using electrons.
6. Equalize electrons.
7. Add half-equations and cancel.

Example: manganate(VII) oxidizes iron(II) in acid.

Reduction:

```text
MnO4-(aq) + 8H+(aq) + 5e- -> Mn2+(aq) + 4H2O(l)
```

Oxidation:

```text
Fe2+(aq) -> Fe3+(aq) + e-
```

Multiply iron half-equation by 5:

```text
5Fe2+(aq) -> 5Fe3+(aq) + 5e-
```

Overall:

```text
MnO4-(aq) + 8H+(aq) + 5Fe2+(aq) -> Mn2+(aq) + 4H2O(l) + 5Fe3+(aq)
```

## Chapter 8: Chemical Reactions and Energy

## Enthalpy Core

Definition:

```text
Delta H = enthalpy of products - enthalpy of reactants
```

Exothermic:

```text
Delta H < 0
Products have lower energy than reactants.
Heat is released.
```

Endothermic:

```text
Delta H > 0
Products have higher energy than reactants.
Heat is absorbed.
```

Calorimetry:

```text
q = mc Delta T
```

Where:

```text
q = heat energy / J
m = mass of solution / g
c = specific heat capacity, usually 4.2 J g^-1 C^-1 for aqueous solution
Delta T = temperature change / C
```

Molar enthalpy:

```text
Delta H = -q / number of moles reacted
```

Negative sign:

- temperature rises: reaction releases heat, so Delta H is negative
- temperature falls: reaction absorbs heat, so Delta H is positive

### Enthalpy of Neutralization

Ionic equation:

```text
H+(aq) + OH-(aq) -> H2O(l)
```

Typical value for strong acid and strong alkali:

```text
about -57 kJ mol^-1
```

DSE focus:

- weak acids/alkalis have less exothermic values because energy is used for ionization

### Enthalpy Cycle / Hess's Law

Hess's law:

```text
The enthalpy change of a reaction is independent of the route taken.
```

Basic cycle:

```text
Reactants -------- Delta H target --------> Products
    \                                      /
     \                                    /
      Delta H1                    Delta H2
       \                                /
        v                              v
              Common intermediate
```

Formula:

```text
Delta H target = Delta H1 - Delta H2
```

Combustion cycle example:

Target:

```text
C(s) + 2H2(g) -> CH4(g)
```

Combustion equations:

```text
C(s) + O2(g) -> CO2(g)                         Delta Hc[C]
H2(g) + 1/2O2(g) -> H2O(l)                     Delta Hc[H2]
CH4(g) + 2O2(g) -> CO2(g) + 2H2O(l)            Delta Hc[CH4]
```

Cycle calculation:

```text
Delta H formation of CH4
= Delta Hc[C] + 2Delta Hc[H2] - Delta Hc[CH4]
```

Formation cycle example:

For:

```text
Reactants -> Products
```

Use:

```text
Delta H reaction = sum Delta Hf(products) - sum Delta Hf(reactants)
```

Bond enthalpy estimate:

```text
Delta H = sum bond enthalpies of bonds broken - sum bond enthalpies of bonds formed
```

DSE common mistakes:

- wrong sign when reversing equation
- forgetting to multiply Delta H when multiplying equation
- mixing formation and combustion cycle direction

## Chapter 9: Rate of Reaction

### Rate Definition

```text
Rate = change in amount of reactant or product / time
```

For gas volume:

```text
Rate = change in gas volume / time
```

Graph:

- steeper gradient means faster rate
- gradient becomes zero when reaction finishes

Equation:

```text

```

Ionic equation:

```text
CaCO3(s) + 2H+(aq) -> Ca2+(aq) + CO2(g) + H2O(l)
```

Factors affecting rate:

- concentration
- temperature
- surface area
- catalyst

```text
Reaction occurs when particles collide with energy equal to or greater than activation energy and correct orientation.
```

Teaching statement:

```text
A catalyst provides an alternative reaction pathway with lower activation energy.

```


## Chapter 10: Chemical Equilibrium

### Dynamic Equilibrium

Definition:

```text
Dynamic equilibrium is reached in a closed system when the forward and reverse reaction rates are equal.
```

Important:

```text
Concentrations remain constant, but reactions continue.
```

Example:

```text
N2O4(g) <=> 2NO2(g)
```

### Equilibrium Constant

For:

```text
aA + bB <=> cC + dD
```

```text
Kc = [C]^c [D]^d / ([A]^a [B]^b)
```

Solids and pure liquids are omitted from Kc expressions.

### Le Chatelier's Principle

Statement:

```text
When a system at equilibrium is disturbed, the system shifts to oppose the change.
```

Concentration:

- adding reactant shifts equilibrium to products
- removing product shifts equilibrium to products

Pressure:

- increasing pressure favors side with fewer gas molecules
- decreasing pressure favors side with more gas molecules

Temperature:

- increasing temperature favors endothermic direction
- decreasing temperature favors exothermic direction

Catalyst:

```text
A catalyst does not change equilibrium position.
It increases both forward and reverse rates equally.
```

## Industrial Chemistry

### Haber Process

Equation:

```text
N2(g) + 3H2(g) <=> 2NH3(g)
Delta H < 0
```

Conditions:

```text
Iron catalyst
About 450 C
High pressure, about 200 atm
```

Equilibrium:

- lower temperature gives higher yield because reaction is exothermic
- higher temperature gives faster rate
- higher pressure gives higher yield because fewer gas molecules on product side
- catalyst increases rate but does not change yield

DSE compromise:

```text
Industrial conditions balance yield, rate, cost and safety.
```

### Contact Process

Key equation:

```text
2SO2(g) + O2(g) <=> 2SO3(g)
Delta H < 0
```

Conditions:

```text
V2O5 catalyst
About 450 C
About 1-2 atm
```

Absorption:

```text
SO3(g) + H2SO4(l) -> H2S2O7(l)
H2S2O7(l) + H2O(l) -> 2H2SO4(aq)
```

DSE focus:

- SO3 is absorbed in concentrated sulfuric acid to form oleum
- direct reaction of SO3 with water is too violent

## Material Chemistry

### Condensation Polymerization

General:

```text
monomers join with elimination of small molecule, often H2O or HCl
```

Polyester example:

```text
diol + dicarboxylic acid -> polyester + water
```

Nylon example:

```text
diamine + dicarboxylic acid derivative -> polyamide + small molecule
```

Cross-linking:

```text
More cross-linking -> harder, more rigid, less flexible
```

## Example Virtual Lab Teaching Card

Experiment:

```text
Reaction of calcium carbonate with hydrochloric acid
```

Aim:

```text
To investigate rate of reaction by measuring carbon dioxide volume.
```

Equation:

```text
CaCO3(s) + 2HCl(aq) -> CaCl2(aq) + CO2(g) + H2O(l)
```

Ionic equation:

```text
CaCO3(s) + 2H+(aq) -> Ca2+(aq) + CO2(g) + H2O(l)
```

Particle explanation:

```text
Higher acid concentration gives more H+ particles per unit volume, causing more frequent successful collisions with CaCO3 surface.
```

Observation:

```text
Bubbles form and gas syringe plunger moves outward.
```

Graph:

```text
CO2 volume against time.
Initial gradient gives initial rate.
```

Safety:

```text
Hydrochloric acid is irritant. Wear goggles.
```

DSE focus:

```text
Explain graph gradient and collision theory.
```
